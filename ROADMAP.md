# Poly Bag Calculator Roadmap

## Phase 1: Stability and Trust

Goal: make the app feel dependable on phone and desktop before adding bigger features.

- improve update detection and background refresh behavior
- show clear online or offline status
- show clear system sync status
- reduce confusion around installed app versions
- keep the calculator responsive and predictable

Status:

- in progress

## Phase 2: UI and Branding

Goal: make the calculator feel more polished and recognizable.

- refine spacing and typography
- improve section hierarchy
- add stronger branding touches
- improve result card presentation
- make status and update messages feel more native

## Phase 3: Faster Pricing Workflow

Goal: make daily use faster for repeated calculations.

- add quick presets
- add last-used values or saved defaults
- add copy result option
- add better summary of the current bag calculation

Status:

- in progress

## Phase 4: Business Tools

Goal: expand the app from calculator to lightweight sales tool.

- customer details
- quotation workflow
- invoice workflow
- export or share options
- notes per order

## Phase 5: Product Expansion

Goal: support more packaging types and future pricing rules.

- more bag categories
- custom formulas
- optional material presets
- catalog-like product selection
